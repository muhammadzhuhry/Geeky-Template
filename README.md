# Geeky-Template

**Geeky** is a template created by using bootstrap framework, that can be used for many purposes, like resume, portfolio, personal, company and many more.

Feature
----------
 - 8 Main Layouts
 - 2 Color Backgrounds
 - One Pages
 - Google Font
 - Cool animation
 - Fully Responsive
 - Smooth Scrolling
 - Font Awesome
 - Text Rotator
 - W3C Validated
 - And Many more
